 CTRL+ALT+T - команда открытия терминала
 CTRL+SHIFT+C - копирование в терминале
 CTRL+SHIFT+V - вставка скопированного текста в терминале
 CTRL+L - очистка терминала от лишнего текста
 ~ - символ домашней дериктории(папки)
 / - символ корневой директории(папки)
 
 .. - обозначение предыдущей директорию 
 . - обозначает текущую директорию
 
1. ls(list) -> Выводит список всех директорий и файлов в текущей директории(рабочая)
   ls -a -> выводит список скрытых и обычных директорий и файлов
   ls -l -> выводит список директорий и файлов с подробной информацией 
   ls -la -> подробный список скрытых и обычных директорий и файлов 
   

2. cd(change directory) -> меняет текущую директорию на указаную
  cd <название директории>
3. mkdir(make directory) -> создание указаной директории
  mkdir <директория>
4. touch - создает указаный файл
  touch <name_of_file>.<extension(расширение(txt,docs,py))>
  touch . - создает скрытый файл 
  touch <.name_of_file> 
5. pwd (print working directory) -> выводит путь до текущей директории

6. rmdir(remove directory) -> Удаляет указанную директорию (не может удалить не пустую директорию)
   rmdir <name_of_directory>
7. rm(remove) -> удаление файла
   rm <name_of_file>
   rm -rf -> принудительное удаление файла или директории
8. mv(move) -> перемещает или переименовывает файл либо директорию,в зависимости от указанного второго аргумента
   mv <файл_директория> <новое_имя -- директория/>
9. cat -> выводит содержимое указанного файла
   cat <name_of_file>
10. nano -> открывает графический интерфейс в терминале
для редактирования файла
*если нет указанного файла,то создасться новый файл с таким именем
  nano <name_of_file>   
11. echo -> команда для быстрой записи данных через терминал
   echo 'текст для записи' > название файла 
12. history -> показывает кол-во команд

13. cp(copy) -> Команда для копирования файла в нужную директорию
  cp <file_to_copy> <path>
